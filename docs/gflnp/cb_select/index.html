CB 自选计划

说明

  • 本文档记录 CB 武器自选计划内容, 包括武器池、自选记录等
  • 自选计划以最长游玩时间的用户名为权限判断
  • 对于 5 星人形, 拥有 1 票选择权
  • 对于 1 日人形, 拥有 2 票选择权
